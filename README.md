igo
===

JavaおよびCommon Lispで実装されたミニマムな形態素解析器

辞書および出力はほぼ[MeCab](http://taku910.github.io/mecab/)互換

- [Java版ドキュメント](https://igo.osdn.jp/)
- [Common Lisp版ドキュメント](https://igo.osdn.jp/cl-igo.html)
